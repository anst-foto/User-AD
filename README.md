## Описание:
Для работы с данным PowerShell-скриптом необходимо подготовить csv-файл - *user.csv* с разделителями ";". В данном файле первой строкой должны идти заголовки столбцов.
### Заголовки:
* **Login** - логин
* **Password** - пароль
* **LastName** - фамилия
* **FirstName** - имя
* **MiddleName** - отчество
* **Department** - отдел
* **Title** - должность
